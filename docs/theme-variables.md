## Keyboard light / dark mode variables:
These variables can be adjusted to customize the modes.
***

### Light mode:
```css
:root {
    --keyboard-background-light: #eeeeee url("/public/assets/ui-bg_highlight-soft_100_eeeeee_1x100.png") 50% top repeat-x;
    --keyboard-border-color-light: #dddddd;
    --keyboard-button-activeButton-background-light: #726e6e;
    --keyboard-button-activeButton-color-light: #ffffff;
    --keyboard-button-background-light: #f6f6f6 url("/public/assets/ui-bg_glass_100_f6f6f6_1x400.png") 50% 50% repeat-x;
    --keyboard-button-border-color-light: #cccccc;
    --keyboard-button-color-light: #000000;
    --keyboard-button-hover-border-color-light: #fbcb09;
    --keyboard-button-hover-color-light: #fdf5ce;
    --keyboard-font-color-light: #000000;
    --keyboard-input-background-light: #eeeeee url("/public/assets/ui-bg_highlight-soft_100_eeeeee_1x100.png") 50% top repeat-x;
    --keyboard-input-border-bottom-light: #726e6e;
    --keyboard-placeholder-font-color-light: #000000;
    --keyboard-select-option-background-color-light: #ececec;
}
```

***

### Dark mode:
```css
:root {
    --keyboard-background-dark: #000000 url("/public/assets/ui-bg_inset-soft_25_000000_1x100.png") 50% bottom repeat-x;
    --keyboard-border-color-dark: #dddddd;
    --keyboard-button-activeButton-background-dark: #ff970f;
    --keyboard-button-activeButton-color-dark: #ffffff;
    --keyboard-button-border-color-dark: #a4a4a4;
    --keyboard-button-background-dark: #555555 url("/public/assets/ui-bg_glass_20_555555_1x400.png") 50% 50% repeat-x;
    --keyboard-button-color-dark: #ffffff;
    --keyboard-button-hover-border-color-dark: #59b4d4;
    --keyboard-button-hover-color-dark: #0078a3;
    --keyboard-font-color-dark: #ffffff;
    --keyboard-input-background-dark: #000000 url("/public/assets/ui-bg_inset-soft_25_000000_1x100.png") 50% bottom repeat-x;
    --keyboard-input-background-color-dark: #000000;
    --keyboard-input-border-bottom-dark: #726e6e;
    --keyboard-placeholder-font-color-dark: #ffffff;
    --keyboard-select-option-background-color-dark: #726e6e;
}
```
